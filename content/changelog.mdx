# Changelog

Stay up to date with the latest features, improvements, and fixes.

## v2.4.0 — 2025-01-15

### Features
- **Portfolio Cockpit** — new dashboard for portfolio-wide visibility
- **AI-powered scenario modeling** — ask "what if" questions naturally
- **Whale curve visualization** — see profit distribution graphically

### Improvements
- Faster engagement loading (50% improvement)
- Enhanced export options for reports
- Mobile-responsive improvements

### Fixes
- Fixed margin calculation rounding errors
- Corrected timezone handling in time entries
- Resolved duplicate notification issues
