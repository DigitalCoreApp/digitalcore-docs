# Profile Settings

Update your personal information and notification preferences.

---

## Personal Information

### Display Name, Email, Avatar

Update your display name from **Settings → Profile**. Your name appears in:

- Navigation
- Comments
- Reports

Upload a profile photo (avatar) — it displays next to your name throughout the application.

Your email address is shown but changes go through the security flow.

---

## Notification Preferences

### Email Notifications

Configure which events trigger email notifications:

| Notification | Description |
|--------------|-------------|
| **Check-in reminders** | Upcoming and overdue check-ins |
| **SLA breach alerts** | When performance thresholds are crossed |
| **Governance updates** | Decision workflow changes |
| **AI-generated insights** | New AI recommendations |

### In-App Notifications

Toggle in-app notifications for real-time alerts within the DigitalCore interface. In-app notifications appear in the notification bell in the top navigation.
