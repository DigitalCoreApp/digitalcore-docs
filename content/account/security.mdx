# Security

Protect your DigitalCore account with strong security practices.

## Password requirements

Passwords must include:
- Minimum 12 characters
- Uppercase and lowercase letters
- At least one number
- At least one special character

## Two-factor authentication (2FA)

Enable 2FA for additional security:

1. Go to **Settings → Security**
2. Click **Enable 2FA**
3. Scan QR code with authenticator app
4. Enter verification code
5. Save backup codes

Supported apps:
- Google Authenticator
- Authy
- 1Password
- Microsoft Authenticator

## Single sign-on (SSO)

Enterprise plans support SSO with:
- Okta
- Azure AD
- Google Workspace

Contact support to configure SSO.

## Session management

View and manage active sessions:

1. Go to **Settings → Security → Sessions**
2. See all logged-in devices
3. Click **Revoke** to log out a device

## API key security

- Generate keys at **Settings → API**
- Keys inherit your permissions
- Revoke keys immediately if compromised
- Use separate keys for different integrations

## Security best practices

- ✅ Enable 2FA
- ✅ Use a password manager
- ✅ Review sessions regularly
- ✅ Rotate API keys periodically
- ✅ Use SSO if available
