# API Quickstart

Get up and running with the DigitalCore API quickly.

## Step 1: Get your API key

1. Log into DigitalCore
2. Go to **Settings → API**
3. Click **Generate API Key**
4. Copy and save the key securely

⚠️ API keys are shown once. Store it safely.

## Step 2: Make your first request

Test your API key with a simple request:

```bash
curl -X GET "https://api.digitalcore.app/v1/me" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

You should see your user profile in the response.

## Step 3: List engagements

```bash
curl -X GET "https://api.digitalcore.app/v1/engagements" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Step 4: Get engagement details

```bash
curl -X GET "https://api.digitalcore.app/v1/engagements/{id}" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Using in code

**JavaScript/TypeScript**:
```javascript
const response = await fetch('https://api.digitalcore.app/v1/engagements', {
  headers: {
    'Authorization': `Bearer ${apiKey}`
  }
});
const engagements = await response.json();
```

**Python**:
```python
import requests

response = requests.get(
    'https://api.digitalcore.app/v1/engagements',
    headers={'Authorization': f'Bearer {api_key}'}
)
engagements = response.json()
```
