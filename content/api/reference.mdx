# API Reference

Complete reference for all DigitalCore API endpoints.

## Base URL

```
https://api.digitalcore.app/v1
```

## Endpoints

### Engagements

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/engagements` | List all engagements |
| `POST` | `/engagements` | Create engagement |
| `GET` | `/engagements/{id}` | Get engagement |
| `PATCH` | `/engagements/{id}` | Update engagement |
| `DELETE` | `/engagements/{id}` | Delete engagement |
| `GET` | `/engagements/{id}/financials` | Get financials |
| `GET` | `/engagements/{id}/team` | Get team members |

### People

| Method | Endpoint | Description |

| `POST` | `/people` | Create person |
| `GET` | `/people/{id}` | Get person |
| `PATCH` | `/people/{id}` | Update person |

## Pagination

List endpoints support pagination:

```
GET /engagements?page=2&per_page=50
```

Response includes:
```json
{
  "data": [...],
  "meta": {
    "page": 2,
    "per_page": 50,
    "total": 234,
    "total_pages": 5
  }
}
```

## Filtering

Filter with query parameters:

```
GET /engagements?status=active&client_id=123
```
