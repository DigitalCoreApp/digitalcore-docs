# How do I set up the MCP server?

The DigitalCore MCP server lets AI assistants access your data for natural language queries.

## What is MCP?

Model Context Protocol (MCP) is an open standard that allows AI assistants to securely access external data sources.

With MCP configured:
- Ask Claude about your engagements
- Query margins in GitHub Copilot
- Generate reports through conversation

## Prerequisites

- DigitalCore API key (from Settings → API)
- Compatible AI client (VS Code, Claude Desktop, etc.)

## VS Code Setup

Add to your VS Code settings:

```json
{
  "mcp.servers": {
    "digitalcore": {
      "command": "npx",
      "args": ["-y", "@digitalcore/mcp-server"],
      "env": {
        "DIGITALCORE_API_KEY": "your-api-key"
      }
    }
  }
}
```

## Claude Desktop Setup

Add to `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "digitalcore": {
      "command": "npx",
      "args": ["-y", "@digitalcore/mcp-server"],
      "env": {
        "DIGITALCORE_API_KEY": "your-api-key"
      }
    }
  }
}
```

## Available tools

Once connected, these tools are available:

- `query_engagements` — search and filter engagements
- `query_finance_summary` — get financial metrics
- `check_engagement_health` — health check for specific engagement
- `record_hours` — log time entries
