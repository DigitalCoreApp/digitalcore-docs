# Integrations

DigitalCore integrates with your existing tools to streamline workflows and eliminate data silos.

## Integration options

| Method | Best For |
|--------|----------|
| [MCP Setup](/docs/integrations/mcp-setup) | AI assistants (Copilot, Claude) |
| [Excel Import](/docs/integrations/excel-import) | Bulk data migration |
| [API](/docs/integrations/api-overview) | Custom integrations |

- Toggl

- Asana

**Finance & Accounting**

**HR & Identity**
- BambooHR
- Okta
- Azure AD

**Communication**
- Slack
- Microsoft Teams

## Setting up integrations

1. Navigate to **Settings → Integrations**
2. Find the integration you want
3. Click **Connect**
4. Follow the OAuth or API key flow
5. Configure sync settings
