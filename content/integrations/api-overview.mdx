# API Overview

The DigitalCore API lets you build custom integrations, automate workflows, and extend functionality.

## What can I do with the API?

- **Read data** — engagements, financials, people
- **Create records** — new engagements, time entries
- **Update data** — modify existing records
- **Trigger actions** — run reports, send notifications
- **Webhooks** — receive real-time updates

## Getting started

1. **Get an API key**: Settings → API → Generate Key
2. **Choose your approach**: REST API or MCP
3. **Review authentication**: [Authentication guide](/docs/api/authentication)
4. **Explore endpoints**: [API Reference](/docs/api/reference)

## API basics

**Base URL**: `https://api.digitalcore.app/v1`

**Authentication**: Bearer token in header
```
Authorization: Bearer your-api-key
```

**Response format**: JSON

## Example request

```bash
curl -X GET "https://api.digitalcore.app/v1/engagements" \
  -H "Authorization: Bearer your-api-key" \
  -H "Content-Type: application/json"
```

## Rate limits

- **Standard**: 100 requests/minute
- **Bulk operations**: 10 requests/minute
- **Headers**: `X-RateLimit-Remaining` shows quota
