# How do I import data from Excel?

DigitalCore supports bulk data import from Excel files.

## What can I import?

- **Engagements** — project/contract data
- **People** — team members and roles
- **Time entries** — historical hours
- **Expenses** — direct costs
- **Clients** — customer records

## How do I import data?

1. Navigate to **Settings → Import**
2. Select data type
3. Download the template
4. Fill in your data
5. Upload the completed file
6. Review mapping
7. Confirm import

## Import templates

Templates ensure data maps correctly. Download from the import page or:

- [People template](#)

## Data validation

DigitalCore validates your data before import:

| Check | Description |
|-------|-------------|
| **Required fields** | All mandatory fields present |
| **Data types** | Numbers are numbers, dates are dates |
| **References** | Client IDs exist, roles are valid |
| **Duplicates** | Warns about potential duplicates |

## Error handling

If validation fails:
1. Download the error report
2. Fix issues in your spreadsheet
3. Re-upload

## Best practices

- Start with a small test import
- Use the exact template format
- Consistent date formats (YYYY-MM-DD recommended)
- Review before confirming
