# Configuration: Bulk Upload & Integrations

Import entities and data in bulk, and connect to external systems.

---

## Bulk Upload Page

### Supported Entity Types

Import the following entities in bulk:

| Entity Type | Description |
|-------------|-------------|
| **Services** | Service definitions |
| **Engagements** | Engagement records |
| **Contracts** | Contract terms and values |
| **Partners** | External partner organisations |
| **Stakeholders** | Contact information |
| **Financial Data** | Historical planned and actual values |

Each type has a downloadable Excel template.

### Upload Flow

1. **Select entity type** and download the template
2. **Upload** your completed file (Excel or CSV)
3. **Review** the staging table — valid rows are green, errors are red with messages
4. **Fix** any errors in the staging view or re-upload a corrected file
5. **Confirm** to write all validated rows to the database

See the [Data Import Guide](/guides/data-import) for detailed instructions and troubleshooting.
