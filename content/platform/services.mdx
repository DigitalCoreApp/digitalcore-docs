# Configuration: Services

Services represent the offerings your organisation delivers to customers.

---

## Services List

### Search and Filtering

The Services page lists all services in your organisation. Use the search bar to find services by name, or filter by category and status.

### Status Indicators

Each service shows a status badge:

| Status | Meaning |
|--------|---------|
| **Active** | Has engagements currently running |
| **Inactive** | No active engagements |
| **Draft** | Still being configured |

---

## Service Detail

### Overview: Name, Description, Category, Owner

The service detail page shows:

- **Name** — Service identifier
- **Description** — What the service provides
- **Category** — Classification for grouping
- **Owner** — Assigned user within your organisation

### Related Engagements

A list of all engagements linked to this service, with their:

- Status
- Health score
- Last check-in date

Click through to any engagement's workspace.

### Related Portfolios

Shows which portfolios include this service. Useful for understanding how a service fits into the broader organisational structure.

---

## Creating and Editing a Service

Click **Create Service** to add a new service. Fill in:

1. Name
2. Description
3. Category
4. Owner

Save to create — you can then create engagements under this service. Edit any field at any time from the service detail page.
