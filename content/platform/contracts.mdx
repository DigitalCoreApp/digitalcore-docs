# Configuration: Contracts

Contracts define the commercial terms, rates, and SLA obligations for your engagements.

---

## Contracts List

View all contracts with:

- Name
- Type
- Total value
- Start/end dates
- Linked engagements

Filter by type (Fixed Price, Time & Materials, Managed Service) or status (Active, Expired, Draft).

---

## Contract Detail

### Overview: Name, Type, Value, Dates

The contract header shows:

| Field | Description |
|-------|-------------|
| **Name** | Contract identifier |
| **Type** | Commercial type (Fixed Price, T&M, Managed Service) |
| **Total Value** | Contract value and currency |
| **Start Date** | When the contract begins |
| **End Date** | When the contract expires |
| **Renewal Date** | When renewal is due (if applicable) |

### Linked Engagements

Lists all engagements operating under this contract. A contract can cover multiple engagements.

### Rate Overrides

The Rates tab shows any role-based rate overrides specific to this contract. Default organisation rates apply for roles without overrides.

See [Rate Cards Guide](/docs/guides/rate-cards) for configuration details.

### SLA Terms

The SLA Terms tab lists all performance terms with financial consequences:

- KPI
- Threshold
- Operator
- Penalty type
- Penalty amount

See [SLA Management Guide](/docs/guides/sla-management) for configuration details.

---

## Creating and Editing a Contract

Click **Create Contract** to add a new contract. Fill in:

1. Name
2. Type
3. Value
4. Dates
5. Link to existing engagements

Add rate overrides and SLA terms as needed.
