# Organisation: Users, Partners & Structure

Manage users, external partners, and your organisational structure.

---

## Users Page

### User List with Status Indicators

View all users in your organisation with:

| Field | Description |
|-------|-------------|
| **Name** | User's display name |
| **Email** | Login email |
| **Role** | Admin, Manager, Operator, or Viewer |
| **Status** | Active, Invited, or Disabled |

### Invite User

Click **Invite User** to send an email invitation:

1. Specify the user's email
2. Assign a role
3. Send invitation

The invitation email contains a link to set up their account.

### Edit User, Reset Password

- Edit a user's role or profile details
- Admins can trigger a password reset email for any user

---

## Pending Invitations Page

### Invitation Status

View all pending invitations:

- Who was invited
- When
- Their assigned role
- Whether the invitation has been accepted, is pending, or has expired

### Resend and Cancel

- Resend an expired or unacknowledged invitation
- Cancel to remove access

## Partners Page

### External Partners List and Detail

Manage external partners — vendors, subcontractors, or collaborators that participate in service delivery.

Each partner has:

- Name
- Description
- Contact information

## Stakeholders Page

### Stakeholder Contacts

Manage stakeholder contacts associated with your projects and engagements:

## Organisational Structure

### Units / Departments

Define your organisation's internal structure:

- Departments

Used for delivery attribution and showback reporting.
