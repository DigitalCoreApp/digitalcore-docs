# How do I create reports?

DigitalCore includes built-in reports and the ability to create custom reports.

## What reports are available?

**Financial Reports**
- Portfolio P&L
- Engagement Profitability
- Revenue by Client/Service
- Variance Analysis

**Operational Reports**
- Utilization Report
- Capacity Forecast
- Time & Attendance
- Resource Allocation

**Performance Reports**
- KPI Scorecard
- SLA Compliance
- Trend Analysis

## How do I run a report?

1. Navigate to **Reports** in the sidebar
2. Select a report
3. Set date range and filters
4. Click **Generate**
5. View on screen or export

## How do I export reports?

Reports export to:
- **PDF** — formatted for printing/sharing
- **Excel** — raw data for further analysis
- **CSV** — data integration

Click **Export** and select format.

## How do I schedule reports?

1. Open a report
2. Click **Schedule**
3. Set frequency (daily, weekly, monthly)
4. Add recipients
5. Save

Reports are emailed automatically.
